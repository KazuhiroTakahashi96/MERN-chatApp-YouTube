# Chat App with MERN Stack

### ローカルでの実行方法

- `.env`ファイルをルートディレクトリに作成し、以下の内容を記述して保存

```
PORT=5000
MONGO_DB_URI=your mongoDB uri
JWT_SECRET=your jwt secret

NODE_ENV=development
```

- ルートディレクトリで、以下の順番でコマンドを実行

```
npm run build
npm run start
↓
// localhost:5000 で開く
```
